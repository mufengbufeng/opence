# Onboarding Specification Delta

## ADDED Requirements
### Requirement: Onboarding checklist
The system SHALL provide a checklist of required setup steps in opence/project.md.

#### Scenario: New developer opens project
- **WHEN** a developer opens opence/project.md
- **THEN** they see a checklist of required setup steps and verification commands

### Requirement: Environment variable reference
The system SHALL document required environment variables with example values.

#### Scenario: Reviewing environment requirements
- **WHEN** a developer reads the onboarding section
- **THEN** required environment variables and examples are listed
